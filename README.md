## 설명

각자 공부한 것을 발표하고 준비했던 자료를 올리는 레포지토리입니다.

### 진행 방식

- 일주일에 한번씩 각자 발표
- 주제는 자유롭게 (os, 네트워크, 표준 등 )
- 노션에는 각자 정리하고 템플릿에 맞게 깃허브에 md 파일로 저장하기
